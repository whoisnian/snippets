# HTTP 请求
后端 Ruby 代码中用到的 HTTP 客户端库主要有：
* [Net::HTTP](https://docs.ruby-lang.org/en/master/Net/HTTP.html): Ruby 标准库自带的 HTTP 客户端，无需安装额外依赖
* [HTTParty](https://github.com/jnunemaker/httparty): 对 Net::HTTP 的封装，调用更简洁，开源项目 [GitLab](https://gitlab.com/gitlab-org/gitlab/-/blob/854658c9a431454ae38007c3ab5ed09505020bbb/Gemfile#L307) 有在使用
* [http.rb](https://github.com/httprb/http): 支持链式调用和流式处理，不基于 Net::HTTP 而是依赖原生的 [llhttp](https://github.com/nodejs/llhttp) 解析器

参考 [OpenTelemetry instrumentation libraries](https://github.com/open-telemetry/opentelemetry-ruby-contrib/blob/main/instrumentation)，常用的 Ruby HTTP 客户端库还有 Faraday、Typhoeus、Excon、RestClient 等，主要特点如下：
* [Faraday](https://github.com/lostisland/faraday): 支持多种适配器，默认使用 Net::HTTP，可以按需切换后端实现
* [Excon](https://github.com/excon/excon): 仅支持 HTTP/1.1 协议，对于 multipart 请求可能需要[手动构造](https://github.com/excon/excon/issues/353#issuecomment-2402815500)请求体
* [HTTPX](https://gitlab.com/os85/httpx): 支持 HTTP/2 和 HTTP/1.X 协议，链式调用，默认启用并发请求特性
* [Typhoeus](https://github.com/typhoeus/typhoeus): 最后版本 1.5.0 发布于2025年8月，前一版本 1.4.1 发布于2023年11月，不再活跃维护
* [Patron](https://github.com/toland/patron): 最后版本 0.13.4 发布于2025年2月，前一版本 0.13.3 发布于2019年5月，不再活跃维护
* [HTTPClient](https://github.com/nahi/httpclient): 最后版本 2.9.0 发布于2025年2月，前一版本 2.8.3 发布于2016年12月，不再活跃维护
* [RestClient](https://github.com/rest-client/rest-client): 最后版本 2.1.0 发布于2019年8月，已停止维护

参考来源：
* [Blog: The state of HTTP clients, or why you should use httpx](https://honeyryderchuck.gitlab.io/2023/10/15/state-of-ruby-http-clients-use-httpx.html)
* [The Ruby Toolbox: HTTP clients](https://www.ruby-toolbox.com/categories/http_clients?display=table&order=score)
* [API test service: httpbin.org](https://github.com/postmanlabs/httpbin)
* [API test service: Postman Echo](https://postman-echo.com)
